# Elysian Highs – Apollo Noni Sales Funnel

## Overview

Elysian Highs is a conversion-focused sales funnel built for the Apollo Noni product ecosystem. The platform is designed to guide visitors through a streamlined customer journey—from initial engagement and lead capture to product purchase and post-purchase interactions.

The system combines modern frontend experiences with checkout and customer workflow functionality to improve conversions, retain leads, and provide a smooth purchasing experience.

---

## Features

### Funnel Pages
- Landing page with promotional messaging and call-to-actions
- Lead capture/opt-in page
- Product sales page
- Checkout/order page
- Thank-you page
- Optional upsell/downsell flow support

### Checkout System
- Dynamic bottle quantity selection
- Cart management functionality
- Persistent cart storage for returning visitors
- Payment workflow integration
- Automatic bonus delivery after successful purchase

### Customer Engagement
- Exit-intent prompts for abandoned visitors
- Abandoned cart recovery workflow
- Customer retention support
- Conversion-focused UI and UX patterns

### Performance & Experience
- Responsive design across desktop, tablet, and mobile devices
- Interactive animations and smooth transitions
- Structured and scalable project architecture
